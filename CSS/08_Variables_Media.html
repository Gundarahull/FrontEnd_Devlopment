Media Query for Responsive Design.....

Here are some common breakpoints in CSS:
  1. sm (small): min-width: 640px
  2. md (medium): min-width: 768px
  3. lg (large): min-width: 1024px
  4. xl (extra-large): min-width: 1280px
  5. 2xl (2x-large): min-width: 1536px
Responsive Content
@media (min-width: 640px), it means only screens that are 640 pixels wide or larger will use this style.
@media (max-width: 640px), it means only screens that are 640 pixels wide or smaller will use this style.
